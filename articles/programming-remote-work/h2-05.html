RUNTEQは、Webエンジニア転職に特化した実践型のプログラミングスクールです。 学習中心のスクールと違い、現役エンジニアによるコードレビューを受けながらチーム開発を体験できます。 1,000時間規模のカリキュラムを9か月で消化する設計で、現場に近い開発スキルが身につきます。 在宅のプログラミング副業でも通用する実装力を身につけられるため、卒業直後から案件に挑戦しやすくなります。 教育訓練給付制度の対象者が所定条件を満たした場合、受講料の一部が給付される可能性があります。適用条件や支給上限は公式サイトで要確認です。 RUNTEQの詳細はこちら

TechElite（テックエリート）

項目 内容
受講料 LP制作コース 330,000円 / HP制作コース 380,000円
期間 8週間〜（習得度には個人差あり）
運営 StockSun株式会社
卒業特典 案件紹介あり
「上司と部下」形式で学べるTechEliteは、実務型カリキュラムのプログラミングスクールです。 独学型のスクールと違い、課題提出→現役エンジニアのレビュー→改善のサイクルを8週間で繰り返します。 在宅のプログラミング副業で需要が高いLP制作とHP制作を、実案件に近い環境で磨けるコース構成です。 卒業後に案件紹介を受けられる制度もあり、副業デビューの初速を上げやすい仕組みになっています。 TechEliteの詳細はこちら
