# feedback_techgym.md — テックジム扱いFB蓄積

## 2026-04-03：フリーランスプログラマー記事FBより

### 1. テックジム料金表示時、システム決済手数料110円を併記する
service-info.mdの料金（入会金33,000円・月額22,000円）に加え、別途システム決済手数料110円がかかる。料金記載時は必ず3点セットで併記する。
- 入会金33,000円（税込）
- 月額22,000円（税込）
- 別途システム決済手数料110円

「追加料金がかからない」「月額だけで済む」等の表現は使用禁止。

## 2026-04-05：テックジム固有ルール追加FB

### 2. テックジムのCTAにrel="nofollow"を絶対に付けない
テックジムは自社サービスのため、rel="noopener"のみ。rel="nofollow"を付けると自社リンクにnofollowが付く事故になる。

### 3. H2タイトルにメインKWを入れる
H2タイトルにKWが入っていないケースが発生。テックジムが関連するH2に限らず、全H2のタイトルに記事のメインKWまたは複合KWを含める。
**NG:** 「独立前に必要な実務経験」
**OK:** 「フリーランスのプログラマーに必要な実務経験」

### 4. 全H3に最低1つのul/liまたは表を入れる
テキストだけのH3は読みにくい。全H3に最低1つのul/li（箇条書き）またはテーブルを含める。
