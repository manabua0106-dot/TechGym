# 禁止語・変換ルール

## 完全禁止（検出されたら即修正）
| 禁止語 | 修正方法 |
|--------|----------|
| こと（名詞化） | 動詞で書き直す |
| として | 具体的な表現に書き直す |
| という | 直接記述する |
| といった | 「〜や〜など」に置き換える |
| これ / その / この / そうした（指示語） | 具体的な名詞で書き直す。「この記事では」「そのまま」は許容 |
| 体言止め（「〜を実施。」「〜に対応。」） | 「〜しています。」「〜です。」「〜できます。」で締める |

## AI臭い語彙（検出されたら即修正）2026-04-05追加
| NG | OK | 理由 |
|----|-----|------|
| 設計（比喩）「働き方を設計」等 | 「決める」「組み立てる」「作る」 | AIっぽい |
| 解放「通勤から解放」等 | 「なくなる」「不要になる」 | AIっぽい |
| 有効「〜として有効です」 | 具体的な結果で書き直す | AIっぽい抽象語 |
| 移行がスムーズ | 「案件が途切れにくくなります」等、具体的な結果 | AIっぽい抽象表現 |
| ルートを描きやすい / パスを描きやすい | 事実だけ書く | 意味不明な修飾 |
| 最大化「手取りを最大化」等 | 「手取りが増える」 | AIっぽい |
| 手段に向いています | 「〜できます」等、読者が得る結果に変える | AI定型文 |
| 経路 | 「方法」 | ナビのような不自然な語 |
| 環境が整っています / 環境です（1セクション内2回以上） | 1回に減らし別表現に | 重複が目立つ |

## 表記統一ルール 2026-04-05追加
| 項目 | ルール |
|------|--------|
| 「方」（人を指す用法） | 全て「人」に統一。「〜の方は」→「〜の人は」 |
| 語尾の重複 | 同じ語尾（「進められます」等）が記事全体で3回以上出たら別表現に差し替え |
| 同一語の1セクション内重複 | 同じ名詞・動詞を1セクション内で2回以上使わない |
| 括弧 | ()は半角、？！は全角に統一 |

## 景表法リスク表現（検出されたら即修正）
| NG | OK |
|----|-----|
| コスパの良さ（根拠なし） | 具体的な価格で記載 |
| 最安クラス（根拠なし） | 具体的な価格で記載 |
| 圧倒的（根拠なし） | 具体的な数字・比較で記載 |
| 大幅割引（比較対象なし） | 「通常価格○○円のところ△△円」 |
| 低価格（根拠なし） | 具体的な価格で記載 |
| No.1（根拠なし） | 調査機関名・条件を明記できる場合のみ |

## 制限あり
| 表現 | ルール |
|------|--------|
| 「〜い。」で終わるアドバイス文 | 「〜です。」「〜できます。」で締める。「〜いです。」はOK |
| 「」（カギ括弧） | 会話文・固有名詞・読者心理描写のみ。強調目的では使わない |
| おすすめです | 記事内3回以内 |
