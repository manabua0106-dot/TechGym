## このファイルの役割

このファイルは **lint.sh / iron-rule-checker が grep するための語彙テーブル** です。
ルールの解釈・背景は `writing-manual.md` が正本。本ファイルはあくまで検査用の実装層。

矛盾が生じた場合は writing-manual.md が優先されます。

## レイヤー1：完全禁止（検出されたら即修正）

| 禁止語 | 修正方法 |
|---|---|
| として | 具体的な表現に書き直す |
| という | 直接記述する |
| といった | 「〜や〜など」に置き換える |
| これ / その / この / そうした（指示語） | 具体的な名詞で書き直す |
| 体言止め（「〜を実施。」「〜に対応。」等） | 「〜しています。」「〜です。」「〜できます。」で締める |

### 指示語の例外（許容）

- 「この記事では」→ OK
- 「そのまま」→ OK（副詞的用法）

### ⚠️ 2026-04 改訂で除外

以下は過去に完全禁止だったが、v1.2 から**使用OK**：

- **こと**（名詞化）→ 自然な日本語として必要な場面がある
- **〜しましょう / 確認しましょう** → feedback_writing.md 学び1 を正として採用。通常文の行動喚起として OK

| 語句 | 許容される文脈 | 制限 |

## 景表法リスク表現（完全禁止・根拠なしの使用）

| NG | OK |
|---|---|
| コスパの良さ（根拠なし） | 具体的な価格で記載 |
| 最安クラス（根拠なし） | 具体的な価格で記載 |
| 圧倒的（根拠なし） | 具体的な数字・比較で記載 |
| 大幅割引（比較対象なし） | 「通常価格○○円のところ△△円」 |
| 低価格（根拠なし） | 具体的な価格で記載 |
| No.1（根拠なし） | 調査機関名・条件を明記できる場合のみ |
| 格安（根拠なし） | 具体的な価格で記載。比較対象・条件が必要 |

## 制限あり

| 表現 | ルール |
|---|---|
| 「〜い。」で終わるアドバイス文 | 「〜です。」「〜できます。」「〜してください。」で締める。「〜いです。」はOK |
| 「」（カギ括弧） | 会話文・固有名詞・読者心理描写のみ。強調目的では使わない |
| おすすめです | 記事内3回以内 |
