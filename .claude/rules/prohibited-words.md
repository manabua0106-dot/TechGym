# 禁止語・変換ルール

## 完全禁止（検出されたら即修正）
| 禁止語 | 修正方法 |
|--------|----------|
| こと（名詞化） | 動詞で書き直す |
| として | 具体的な表現に書き直す |
| という | 直接記述する |
| といった | 「〜や〜など」に置き換える |
| これ / その / この / そうした（指示語） | 具体的な名詞で書き直す。「この記事では」「そのまま」は許容 |
| 体言止め（「〜を実施。」「〜に対応。」） | 「〜しています。」「〜です。」「〜できます。」で締める |

## 比喩表現（検出されたら即修正）
| NG | OK |
|----|-----|
| コストが跳ね上がる | 費用が高くなる |
| 費用がふくらむ | 費用が予算を超えやすい |
| 広げる（比喩） | 増やす・伸ばす |
| 鍵（比喩） | 重要なポイント |
| 分かれ目 | 判断基準 |

| さまざまなメリットがあります | 前置きなしで本題に入る |
| ぜひ参考にしてください（2回以上） | 導入文末尾1回のみ |
| 〜への第一歩 | 具体的な行動で締める |
| 〜を両立させています | 〜に定評があります |
| 〜の強みです | 具体的な特徴で書き直す |
| 〜も魅力のひとつです | 〜もメリットです / 〜も見逃せません |
| 機能もあります | 機能名を主語にして具体的に書く |

| ポイントです | 具体的メリットに置き換え |

## 景表法リスク表現（検出されたら即修正）
| NG | OK |
|----|-----|
| コスパの良さ（根拠なし） | 具体的な価格で記載 |
| 最安クラス（根拠なし） | 具体的な価格で記載 |
| 圧倒的（根拠なし） | 具体的な数字・比較で記載 |
| 大幅割引（比較対象なし） | 「通常価格○○円のところ△△円」 |
| 低価格（根拠なし） | 具体的な価格で記載 |
| No.1（根拠なし） | 調査機関名・条件を明記できる場合のみ |
| 格安（根拠なし） | 具体的な価格で記載。比較対象・条件が必要 |
| 最も（根拠なし） | 「条件次第では〜しやすい」等に書き換え |
| 最短（成果保証） | 「○週間のカリキュラムが用意されています。個人差があります」 |
| より早く / より安く（比較対象なし） | 比較対象・条件を明示。不明なら削除 |
| 効率よく（根拠なし） | 具体的な仕組みや方法を説明する |
| 選ばれています（根拠なし） | 「〜に向いています」に書き換え |
| 実質○○円（条件不明） | 制度名・対象者要件・上限額を併記。不明なら「公式サイトで要確認」 |
| 最大○○%給付/割引（条件不明） | 制度区分・対象要件・上限を明記。「条件は公式サイトで要確認」を付記 |

## 制限あり
| 表現 | ルール |
|------|--------|
| 「〜い。」で終わるアドバイス文 | 「〜です。」「〜できます。」で締める。「〜いです。」はOK |
| 「」（カギ括弧） | 会話文・固有名詞・読者心理描写のみ。強調目的では使わない |
| おすすめです | 記事内3回以内 |
