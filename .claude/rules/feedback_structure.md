## このファイルの役割

Manabuさんからの**新しい構成に関する指摘**を日付付きで一時記録する場所です。

### 運用フロー

```
1. 新しい指摘を受ける
   ↓
2. このファイルに日付付きで追記
   ↓
3. 同じ指摘が3回以上 or 恒久ルール化が妥当と判断
   ↓
4. writing-manual.md に昇格
   ↓
5. このファイルから該当項目を削除（SSOT 維持のため二重管理回避）
```

### 記録フォーマット

```markdown
## YYYY-MM-DD: 指摘のタイトル

### 学びN: 簡潔なルール名

- NG: 具体的な NG 例
- OK: 具体的な OK 例
- **ルール**: ルールの要約
- **適用範囲**: 全記事 / 特定セクション / 特定記事タイプ
- **昇格候補**: writing-manual.md の該当セクション
```
