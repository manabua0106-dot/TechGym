# TechGym 鉄のルール（絶対遵守）
# このファイルの内容は例外なく遵守する。Manabuさんの許可なく修正してよい。

## 1. pタグ禁止
HTMLに<p>タグを含めない。改行のみで段落を分ける。
<p>も</p>も一切使わない。

## 2. 禁止語ゼロが納品条件
以下の語が1件でも残っていたら納品できない。

| 禁止語 | 検出パターン | 修正方法 |
|--------|------------|----------|
| こと（名詞化） | 「〜できること」「〜なこと」等 | 動詞で書き直す |
| として | 「〜として、」「〜として。」 | 具体的な表現に書き直す |
| という | 「〜ということ」「〜という〇〇」 | 直接記述する |
| といった | 「〜といった」 | 「〜や〜など」に置き換える |
| 指示語 | 「これ」「その」「この」「そうした」 | 具体的な名詞で書き直す |
| 体言止め | 「〜を実施。」「〜に対応。」 | 「〜しています。」「〜です。」で締める |

例外：「この記事では」「そのまま」は許容。

## 3. service-info.mdが唯一の正
テーブル・画像・CTA・アフィリURLは全てservice-info.mdからコピペする。
外部データを混ぜない。手打ちしない。

以下の項目はservice-info.mdと完全一致させる：
- 画像のsrc属性（パス）
- 画像のwp-image-ID（class属性内の数字）
- 画像のwidth・height
- アフィリURLの全パラメータ（a_id, p_id, pc_id, pl_id, a8mat）
- CTAのrel属性（アフィリ→rel="nofollow"、テックジム→rel="noopener"のみ）

## 4. テックジムのCTAにアフィリタグを混入させない
テックジムの紹介セクション内に以下のURLパターンが含まれていたら即削除する：
- af.moshimo.com
- px.a8.net
- gokindler.com

テックジムのリンクにrel="nofollow"を付けない。

## 5. 推測で数値を書かない
確認できない料金・給付金情報・キャンペーン条件は「公式サイトで要確認」と書く。
「〜とされています」「〜と言われています」等の曖昧表現で逃げない。

## 6. 1回の実行で1つのH2だけ書く
記事全文を一気に書かない。1つのH2を書き終えたらチェッカーを通す。
チェックをパスしてから次のH2に進む。

## 7. 比喩表現禁止
物理的な意味と混同しうる比喩表現は使用しない。
NG：「コストが跳ね上がる」「費用がふくらむ」「広げる」「鍵」「分かれ目」
OK：「費用が高くなる」「予算を超えやすい」「増やす」「重要なポイント」「判断基準」
