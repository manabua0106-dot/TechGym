# TechGym Memory — 記憶の索引

## このフォルダの役割
Manabuさんからのフィードバックや、記事制作を通じて得た学びを蓄積する。
次回以降の記事制作で自動的に適用される。

## 重要ルール
- **ここに入れるのは「次の会話でも使える、普遍的な学び」だけ。**
- 今やっている作業の進捗や一時的な情報は入れない（それはarticles/フォルダの仕事）。
- Manabuさんが「許容」「これでいい」「もう指摘しないで」と言った判断は必ずここに記録する。
- 同じフィードバックを二度させない。一度言われたことは学習する。

## 更新手順
1. Manabuさんからフィードバックを受けたら、該当するmemoryファイルに追記する
2. 追記する際は日付と内容を1行で簡潔に書く（例：`2026-04-02: 「しましょう」はTechGymでは許容`）
3. MEMORY.mdの最終更新日を更新する
4. 矛盾するFBが来た場合は、新しい方を採用し、古い方に取り消し線を入れる
