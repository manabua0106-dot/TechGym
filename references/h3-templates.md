# H3テンプレート集（TechGym用）
# writerが執筆時に参照する。各H3タイプごとの型と具体例を収録。

═══════════════════════════════
## 選び方H3のテンプレート
═══════════════════════════════

### 構成
```
[1文目] 結論（このH3で言いたいことの要点）
[2文目] 根拠・理由（なぜそれが重要か）
[3〜4文目] 具体例・補足（数字や具体的な方法）
[最終文] <strong>タグのstrong締め（具体的な行動 or 読者にとっての価値）</strong>
```

### 具体例：「受講形式が自分のライフスタイルに合っているか」
```html
オンライン完結型か通学型か、受講形式を事前に確認してみてください。
仕事をしながら学ぶ方は、平日夜や週末に受講できるスケジュールかどうかが重要です。
通学型のスクールは対面で質問しやすいメリットがありますが、通学時間がかかります。
オンライン型は場所を選ばず学べる反面、自己管理が求められます。
<strong>公式サイトの受講スケジュール欄で、自分の生活リズムに合うか確認してみてください。</strong>
```

### 具体例：「給付金制度に対応しているか」
```html
教育訓練給付金の対象講座なら、受講料の最大70%が国から給付されます。
スクールによって対象コースと非対象コースが分かれているため、事前確認が必要です。
給付金の申請にはハローワークでの手続きが必要で、受講開始の1ヶ月前までに申し込みます。
<strong>給付金を使えば自己負担を大幅に抑えてスキルを習得できます。</strong>
```

## メリットH3のテンプレート
═══════════════════════════════

### 構成
```
[1文目] 結論（このメリットの要点。メインKW/複合KWを含める）
[2文目] 根拠・理由
[3〜4文目] 具体的な効果（読者の生活・キャリアでの変化）
[最終文] <strong>タグのstrong締め（読者にとっての価値）</strong>
```

### 具体例：「仕事と並行して学習を進めやすい」
```html
プログラミングスクールの多くは、仕事と並行して学べるカリキュラムを用意しています。
オンライン教材なら通勤時間や昼休みにも学習を進められます。
自習型のスクールでは決まった時間割がないため、急な残業があっても振替の必要がありません。
<strong>仕事を辞めずにスキルを習得できるため、収入を維持したまま転職準備ができます。</strong>
```

## 注意点H3のテンプレート
═══════════════════════════════

### 構成
```
[1文目] 結論（この注意点の要点。メインKW/複合KWを含める）
[2文目] 失敗パターン（読者のあるあるシーン）
[3〜4文目] 対策（具体的な行動）
[最終文] <strong>タグのstrong締め（対策の効果 or 具体的な行動）</strong>
```

### 具体例：「受講料以外のコストも含めて比較する」
```html
プログラミングスクールの料金を比較するときは、受講料だけでなく入会金・教材費・PCレンタル代も含めた総額で比べてみてください。
受講料が安く見えても、入会金が数万円かかるスクールもあります。
公式サイトの料金ページで「税込総額」と「追加で必要な費用」の両方を確認してください。
<strong>総額で比較すれば、本当にコストを抑えられるスクールが見えてきます。</strong>
```

## サービス紹介H3のテンプレート
═══════════════════════════════

### 構成（テーブルなし・紹介文のみの場合）
```
[段落1] <strong>タグでサービスの特徴キャッチ。1文目に複合KWを含める。特徴は1つに絞る。
[段落2〜3] 具体的な特徴・強み。機能の羅列ではなく読者メリットに変換。
[段落4（任意）] キャンペーン・給付金情報。
[最終行] CTA（cta-templates.mdからコピペ）
```

### 構成（テーブルあり・東京記事の場合）
```
[H2見出し] ■サービス名 ｜キャッチコピー
[画像] wp:imageブロック（service-info.mdの画像を使用）
[紹介文2〜3段落] サービスの概要・特徴・強み
[テーブル] 3行スペック表（場所・料金・学べる言語）
[CTA下テキスト] 「〜の詳細が知りたい方は下記をご覧ください。」
[VKボタンCTA] cta-templates.mdからコピペ
```

### 具体例（テーブルなし）：
```html
<a href="https://af.moshimo.com/af/c/click?a_id=5222640&amp;p_id=2717&amp;pc_id=6139&amp;pl_id=39604" rel="nofollow">RUNTEQ</a>は、<strong>Webエンジニアへの転職に特化したプログラミングスクール</strong>です。

カリキュラムは実務に近い開発環境で進めるため、現場で通用するスキルが身につきます。
受講中にポートフォリオを複数作成でき、転職活動で実績としてアピールできます。

教育訓練給付金の対象講座で、受講料の最大80%が給付されます。

<a href="https://af.moshimo.com/af/c/click?a_id=5222640&amp;p_id=2717&amp;pc_id=6139&amp;pl_id=39604" rel="nofollow">RUNTEQ 公式サイト</a>
```

### 書き出しパターン（最低3種類用意する）
パターン①：「<a>サービス名</a>は、<strong>〜特徴〜</strong>のプログラミングスクールです。」
パターン②：「<strong>〜特徴〜</strong>を提供する<a>サービス名</a>は、」
パターン③：「<a>サービス名</a>の最大の特徴は、<strong>〜</strong>です。」
パターン④：「月額〇〇円から受講できる<a>サービス名</a>は、」（安さ特化記事）

## FAQ H3のテンプレート
═══════════════════════════════

### 構成
```
[H3] Q. 質問文
[1文目] 回答の結論
[2〜3文目] 補足・具体的な情報
```

### ルール
- 本文で既に扱った内容の焼き直しは禁止
- サジェスト・知恵袋・PAA・競合のFAQから、本文で未対応のニーズを選ぶ

### 具体例：
```html
<h3>Q. 未経験からでもプログラミングスクールに通えますか？</h3>
はい、多くのスクールが未経験者向けのカリキュラムを用意しています。
テックジムやRUNTEQでは、受講生の9割以上がプログラミング未経験からスタートしています。
無料体験やカウンセリングを受ければ、自分のレベルに合うか事前に確認できます。
```

## まとめH2のテンプレート
═══════════════════════════════

### 構成
```
[1文目] <strong>記事全体の結論を1文で要約</strong>
[2文目] <strong>読者にとっての最も重要なポイント</strong>
[3文目] 具体的な次のアクション（無料カウンセリング・体験レッスン等）
```

### 具体例：
```html
<strong>プログラミングスクールは料金・受講形式・サポート体制が大きく異なるため、自分の目的に合ったスクールを選ぶのが重要です。</strong>
<strong>給付金制度を活用すれば自己負担を大幅に抑えてスキルを習得できます。</strong>
まずは気になるスクールの無料カウンセリングに参加して、カリキュラムや雰囲気を確認してみてください。
```
